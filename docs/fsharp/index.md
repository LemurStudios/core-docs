# F# Guide

F# is a cross-platform, open source programming languge for .NET which provides support for functional programming in addition to traditional object-oriented and imperative (procedural) programming.  The Visual F# compiler and tooling are Microsoft's implementation and tooling for the F# programming language, making F# a first-class member of .NET.
